# Guia do GitHub - Como escrever através do VsCode e do Terminal no ReadMe e como salvar no Git
## 1° passo: Criar um Repositório e de um nome a ele
## 2° passo: Definir se ele será público ou privado
## 3° passo: Gerar code e copiar o link 
## 4° passo: Vá para o explorer e abra uma pasta para guardar o repositório criado
## 5° passo: Aperto o botão direito do mouse para abrir o terminal
## 6° passo: Utilize o comando "git clone" e cole o link copiado
## 7° passo: Digite o comando "cd" e o nome do repositório criado (para entrar nele)
## 8° passo: Digite o comando "code ." (abir o code)
## 9° passo: Digite o comando "git add ." 
## 10° passo: Digite "git commit -m" e entre aspas, digite a mudança realizada
## 11° passo: Digite o comando "git push -u originin main" para empurrar o branch para o principal (Lembrar de entrar na conta do git)
